HTML форми та атрибути

form

Атрибути

  • action: вказує URL-адресу для надсилання даних форми.
  • method: - метод передачі даних GET (за замовченням) - безпосередньо в адресному рядку POST - приховано.
  • enctype: кодування даних під час відправки на сервер. Для передачі файлів використовуємо - multipart/form-data Детальніше: https://www.w3schools.com/tags/att_form_enctype.asp

input

Атрибути

  • name - ім'я поля (даних)
  • value - значення поля (дані)
  • autocomplete - дозволяє (за замовченням) / забороняє (значення off) автозаповнення поля
  • required - обо'язково для заповнення
  • maxlength- максимальна кількість символів доступних для вводу
  • placeholder - підпис поля
  • minlength - вимога до мінімальної кількості символів. працює в парі з required
  • checked (для деяких типів) - вибраний за замовченням
  • disabled - поле перестає бути доступним для користування, данні передані не будуть
  • pattern - можливість вказувати регулярні вирази
  • type="" Усі типи https://www.w3schools.com/html/html_form_input_types.asp

label

Атрибути

  • label асоціює текстову мітку з полем форми, що робить форму більш доступною. Аттрибут "for" прив'язує мітку до конкретного input через його id.

textarea > select > option > optgroup

Тег

  • textarea - ввід багаторядкового тексту
  • select — випадаючий список, майже не стилізується
  • option — елемент списку;
  • optgroup — використовується всередині тега select для створення групи опцій.

button

Атрибут type=""

  • "button" - кнопка нічого не робить :)
  • "reset" - обнуляє (скидає) поля вводу у формі. Має знаходитись в середині тегу form.
  • "submit" - (за замовченням) відправляє форму. Має знаходитись в середині тегу form.
